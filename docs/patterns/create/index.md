# Создание выкроек

В **CustomKnits** есть несколько способов создать новую выкройку:

- по **отдельному шаблону** одной детали;
- через **пресеты вещей**, когда сразу создаются перед, спинка, рукав и другие части изделия;
- **вручную**, нарисовав контур полигона прямо на доске.

Создание по шаблону или пресету запускается из боковой панели в разделе **«Выкройки»**.

## Доступные способы создания

### **1. Отдельные выкройки (шаблоны)**
Создаётся **одна** деталь по выбранному типу (например, шапка или шарф).  
Подробнее: **[Создание по шаблону](./template/)**.

---

### **2. Пресеты вещей**
За один шаг создаётся **несколько деталей** изделия: перед, спинка, рукав, воротник.  
Подробнее: **[Создание по пресетам вещей](./preset/)**.

---

### **3. Ручное рисование полигона**
Можно создать выкройку *совершенно любой формы*, рисуя контур на доске.  
Подробнее: **[Создание вручную (полигон)](./create-manual.md)**.

---

## Что выбрать?

- Если нужна **одна деталь** — удобнее начать с шаблона.  
- Если вы делаете **целое изделие** — быстрее использовать пресеты.  
- Если нужна **нестандартная форма** — лучше рисовать вручную.

Это три равноправных способа, и все создают полноценные редактируемые выкройки.
