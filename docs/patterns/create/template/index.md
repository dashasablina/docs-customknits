# Создание выкройки по шаблону

Создание выкроек по пресетам вещей
Этот способ подходит, когда нужна **одна выкройка** по простому шаблону: шапка, шарф и другие базовые формы.

---

## Как открыть шаблоны выкроек

1. В боковой панели в разделе **«Выкройки»** нажмите кнопку **«+»**.  
2. В появившемся окне выберите блок **«Отдельные выкройки»**.

Откроется окно **«Шаблоны выкроек»**:

## Заполнение шаблона

В окне шаблонов:

1. Выберите нужный **тип выкройки** из выпадающего списка (например, шапка).  
2. Введите основные мерки — обязательные поля помечены звёздочкой `*`.  
3. При необходимости включите **расширенный режим**, чтобы увидеть дополнительные настройки.  

После заполнения нажмите **«Создать выкройку»**.

Редактор построит контур детали по заданным меркам и добавит её на общую доску выкроек.

---

## Что происходит после создания

- Новая деталь появится на доске рядом с уже существующими выкройками.  
- Эта деталь сразу станет **активной** — её контур будет выделен более ярким цветом.  
- Дальше вы можете редактировать её как обычную выкройку:
  - добавлять и перемещать точки,
  - менять длину линий,
  - выравнивать и зеркалить части,
  - создавать по ней петли для вязания.

Все изменения сохраняются внутри проекта и будут доступны при следующем открытии файла.

---

## Когда удобно использовать шаблоны

Шаблоны удобны, когда:

- вы хотите быстро получить **базовую форму** детали, а не рисовать её с нуля;  
- нужно многократно строить похожие выкройки с разными мерками (например, одинаковые шапки на разные размеры головы);  
- важна точность по меркам, но детали потом всё равно будут **дорабатываться вручную**.

Если вам нужно сразу построить **несколько деталей одного изделия** (перед, спинка, рукав), используйте **пресеты вещей** — они описаны в следующем разделе.
