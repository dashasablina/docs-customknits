# Что такое выкройка в CustomKnits

Выкройка в редакторе — это набор точек, соединённых отрезками в замкнутый контур.  
Все размеры задаются в сантиметрах.

Сейчас редактор работает только с прямыми сегментами — кривых нет.  
Это не является ограничением: длина отрезков может быть сколь угодно маленькой, поэтому можно приблизится к желаемой кривой линии.

Для вязания этого более чем достаточно.

Все выкройки проекта располагаются в одной рабочей области — это позволяет:

- сравнивать выкройки между собой;  
- выстраивать детали относительно друг друга

## Ограничения текущей версии

- Поворачивать выкройки нельзя. 
- Кривых сегментов нет, но форму можно приближать необходимым количеством коротких отрезков.
