# Удаление выкроек

Выкройку можно удалить двумя способами:

- **Через боковую панель** — выберите выкройку в списке и используйте команду удаления.  
- **Клавишей Delete / Backspace** — когда есть **активная выкройка**, но **ни одна её точка не выделена**.

> Если по выкройке раньше было создано вязание,  
> **удаление выкройки никак не влияет на это вязание** — оно остаётся в проекте.
