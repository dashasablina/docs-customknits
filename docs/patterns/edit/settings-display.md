# Настройки отображения

На доске выкроек можно управлять тем, **что именно отображается** поверх контура:

- сетка;
- длины сторон;
- углы между отрезками.

Все настройки собраны в меню с иконкой **глаза** в левой верхней части рабочей области.

## Шаг сетки

Поле **«Шаг сетки, см»** задаёт расстояние между линиями сетки:

- `5` → клетки 5×5 см;
- крупная сетка удобна для набросков и крупных деталей;
- мелкая — для точной подгонки.

---

## Показ длин сторон

Опция **«Показывать длины сторон»** включает подписи длины вдоль линий контура:

<img
  src={require('/img/pattern/showLength.png').default}
  srcSet={require('/img/pattern/showLength.png').default + " 2x"}
  alt="Отображение длин сторон на выкройке"
/>

- единицы измерения — сантиметры;
- значения обновляются при перемещении точек;
- помогает контролировать соответствие меркам и симметрию.

---

## Показ углов

Опция **«Показывать углы»** добавляет дуги с углом между соседними отрезками:

<img
  src={require('/img/pattern/showAngles.png').default}
  srcSet={require('/img/pattern/showAngles.png').default + " 2x"}
  alt="Отображение углов на выкройке"
/>

Полезно для:

- пройм и окатов;
- вырезов горловины;
- плавных изгибов.

---

## Сброс

Кнопка **«Сбросить настройки»** возвращает:

- шаг сетки;
- показ длин;
- показ углов

к значениям по умолчанию.

Все эти параметры относятся **только к доске выкроек** и не влияют на другие режимы  
(например, доску вязания).
