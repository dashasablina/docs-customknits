# Изменение выкройки

Все выкройки проекта отображаются на **одной общей доске**.  
Это удобно для сравнения форм, размеров и взаимного расположения деталей.

---

## Активная выкройка

Редактируется только **выбранная сейчас выкройка**:

- активная выкройка подсвечивается ярче остальных;
- неактивные детали приглушены, но остаются видимыми;
- все действия с точками и формой относятся именно к активной детали.

Чтобы выбрать выкройку:

- кликните **внутри её контура** на доске, или  
- выберите её в списке **«Выкройки»** в боковой панели.

---

## Редактирование по точкам

Форма выкройки задаётся **точками**, соединёнными линиями.  
С точками можно:

- добавлять новые;
- перемещать существующие;
- удалять лишние;
- выравнивать и зеркалить целые группы.

Подробно это описано в разделе  
👉 **[Редактирование по точкам](./points.md)**.

---

## Что можно настроить

Для удобства работы можно изменить:

- **что показывать на доске** — сетку, длины сторон, углы и т.п.:  
  👉 **[Настройки отображения](./settings-display.md)**  

- **как выглядит интерфейс выкроек** — цвета линий, заливку, точки:  
  👉 **[Настройки интерфейса](./settings-interface.md)**

Эти параметры не меняют саму выкройку — только то, **как она выглядит**.
