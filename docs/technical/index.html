Технические требования

Чтобы приложение CustomKnits работало корректно и стабильно, убедитесь, что ваше устройство и браузер соответствуют минимальным рекомендациям, описанным ниже.

Поддерживаемые устройства​

🖥️ Настольные компьютеры и ноутбуки — полностью поддерживаются.
Приложение разработано для работы на широких экранах с использованием мыши и клавиатуры.

📱 Мобильные устройства — временно не поддерживаются.
Интерфейс требует большого рабочего пространства, поэтому на смартфонах и планшетах возможны ошибки отображения или полная невозможность запуска.

Операционные системы​

CustomKnits работает в современных браузерах на следующих ОС:

  • Windows 10+
  • macOS 11+
  • Linux (при использовании актуальных браузеров)

Поддерживаемые браузеры​

Приложение построено на React и использует современные Web API, поэтому рекомендуется использовать последние версии браузеров.

Поддерживаются:

  • Google Chrome (рекомендуется)
  • Microsoft Edge
  • Mozilla Firefox
  • Safari (на macOS)

⚠️ В режиме «Инкогнито» возможности локального сохранения могут быть ограничены.

Требования к устройству​

  • Размер экрана: от 13″
    (рекомендуемое разрешение — от 1280×720 и выше)
  • Оперативная память: минимум 4 ГБ
  • Процессор: любой современный двухъядерный
  • JavaScript: должен быть включён
  • Интернет: требуется стабильное подключение для загрузки и сохранения проектов

Возможные ограничения​

  • На устаревших ноутбуках работа с большими полотнами может выполняться медленнее.
  • Если JavaScript отключён — приложение не сможет загрузиться.
  • Встроенные браузеры (WebView) и мобильные браузеры пока не поддерживаются.
