# Узоры и заполнение узорами

Чтобы выполнить действие **заполнения узором**, нужен выбранный узор.  
Если узор уже выбран, его миниатюра отображается в панели рядом с выбранной пряжей, плотностью и типом петли:

/>

---

## Что такое узор

**Узор** — это прямоугольный набор петель.  
Он повторяется по всей выделенной области при заполнении узором.

Узор может содержать:

- только типы петель,  
- только пряжу,  
- только плотности,  
- любое сочетание этих свойств,  
- или все три одновременно.

Таким образом, узор может быть:
- цветовым,
- текстурным,
- плотностным,
- комбинированным.

---

## Как применяются свойства узора

Если узор содержит несколько разных пряж, плотностей или типов петель, вы можете выбрать, какие из этих свойств будут применены.  
Для этого существуют **настройки заполнения узором**, доступные из панели действий:

<img
  src={require('/img/knitting/tileSettingsForArea.png').default}
  srcSet={require('/img/knitting/tileSettingsForArea.png').default + " 2x"}
  alt="Настройки затайливания"
/>

Параметры настройки:

- **Переносить пустые клетки из буфера**  
- **Заполнять пустые клетки внутри выделенной области при затайливании**  
- **Изменять типы петель**  
- **Изменять пряжу**  
- **Изменять плотность**

---

## Локальная пряжа узора

Узоры могут содержать свою собственную палитру пряжи.

Если включить опцию **«Использовать пряжу пресета»**, пряжа из узора будет добавлена в список пряж проекта:

<img
  src={require('/img/knitting/selectedStitchPattern.png').default}
  srcSet={require('/img/knitting/selectedStitchPattern.png').default + " 2x"}
  alt="Использование пряжи узора"
/>

Если галочку не ставить, редактор применит цвета из текущей палитры проекта.

---

## Сброс выбранного узора

Кнопка **«Сбросить выбор»** возвращает узор к дефолтному состоянию.

**Дефолтный узор** — это один единственный элемент, построенный из:
- выбранного типа петли,
- выбранной пряжи,
- выбранной плотности.

---

## 2. Выбрать узор из библиотеки

Чтобы выбрать узор из библиотеки, нажмите на превью текущего узора в панели:

/>

Откроется окно с двумя вкладками: **Библиотека узоров** и **Генератор узоров**.  
Выберите вкладку **Библиотека узоров**.

  srcSet={require('/img/knitting/stitchPatternLibrarySets.png').default + " 2x"}

Пока узоров немного, но библиотека будет расширяться.

## 3. Использовать генератор узоров

Вторая вкладка — **Генератор узоров**.  
Генератор позволяет автоматически создавать узоры, изменяя цвета, размеры и стратегию заполнения.

<img
  src={require('/img/knitting/generator.png').default}
  srcSet={require('/img/knitting/generator.png').default + " 2x"}
  alt="Генератор узоров"
/>

В генераторе можно настроить:

- **Цвет А** и **Цвет B**
- **Ширину и высоту узора**
- **Размер превью**
- **Стратегию заполнения**

Плотность в генераторе нужна только для визуализации.

<img
  src={require('/img/knitting/generatorChooseColor.png').default}

Доступные стратегии:

После настройки нажмите **Создать узор**, чтобы сгенерировать новый вариант. Можно сколько угодно раз генерировать узор.

## Предупреждение

Стратегии **«зебра»** и **«пятна жирафа»** пока работают нестабильно и могут давать странные результаты.  
Мы будем развивать генератор, улучшать существующие алгоритмы и добавлять новые варианты, включая генерацию по типам петель.

# Итоги

Узор — это любая область петель, которую можно повторять как орнамент.

Он может задавать рисунок, цвет, тип петель, плотность — или любое сочетание этих свойств.

Узор можно создать вручную, выбрать из библиотеки или сгенерировать автоматически.

При заполнении узором редактор раскладывает узор по выделенной области, повторяя его столько раз, сколько нужно.
