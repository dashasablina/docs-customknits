# Режимы отображения

Сейчас в редакторе доступен один режим отображения — **Схема**.

В режиме *Схема* вязание представлено как плоская сетка, где каждая петля нарисована на своём месте.  
Цвет петли соответствует выбранной пряже, а пустые клетки обозначают отсутствие петель.
