# Пряжа

Пряжа в редакторе отвечает за цвет петель. Для каждой петли известно, из какой она пряжи.
Петля может быть связана только из одной пряжи.

В проекте может быть несколько пряж, но в каждый отдельный момент используется только одна: **выбранная пряжа**.

## Быстрый обзор

- активная пряжа отображается иконкой клубка на панели;
- список пряж позволяет менять порядок, название, цвет и метраж;
- для секционной пряжи можно редактировать секции и включать эффект ручного окрашивания;
- изменения пряжи автоматически применяются к петлям, где она используется.

---

## Выбор активной пряжи

Выбранная пряжа показывается в клубке на панели инструментов.
Чтобы открыть полный список пряж проекта, нажмите на клубок.

<img
  src={require('/img/knitting/yarnList.png').default}

Для каждой пряжи в списке обязательно указаны **название** и **цвет**.
У секционной пряжи иконка разноцветная.

Чтобы переименовать пряжу, нажмите на название.

При наведении на пряжу в списке появляются действия:

<img
  src={require('/img/knitting/yarnOnhover.png').default}
  srcSet={require('/img/knitting/yarnOnhover.png').default + " 2x"}
  alt="Действия с пряжей"
/>

- **копировать** пряжу;
- **удалить** пряжу (если нет петель из этой пряжи);
- **перетаскивать** пряжи для изменения порядка;
- **изменить метраж**;

Порядок в списке влияет только на удобство выбора.

## Цвет пряжи

Чтобы изменить цвет пряжи, нажмите на квадратик цвета.
Для однотонной пряжи сразу появится выбор цвета.

<img
  src={require('/img/knitting/changeYarnColor.png').default}
  srcSet={require('/img/knitting/changeYarnColor.png').default + " 2x"}
  alt="Изменение цвета пряжи"
/>

Для секционной - откроется список секций, где вы сможете выбрать цвет и длину каждой секции отдельно.

Важно: вы можете перетаскивать секции, чтобы поменять их порядок. Это повлияет на итоговый вид пряжи.

<img
  src={require('/img/knitting/changeYarnSectionColor.png').default}
  srcSet={require('/img/knitting/changeYarnSectionColor.png').default + " 2x"}
  alt="Изменение секций пряжи"
/>

Чтобы превратить однотонную пряжу в секционную - нажмите кнопку Сделать секционной в окошке выбора цвета пряжи.
Чтобы превратить секционную пряжу в однотонную - удалите все ее секции.

Для секционной пряжи можно добавить **«ручное окрашивание»**.
Это чисто визуальный эффект, добавляет немного цветового шума.

<img
  src={require('/img/knitting/handDyedYarn.png').default}
  srcSet={require('/img/knitting/handDyedYarn.png').default + " 2x"}
  alt="Ручное окрашивание"
/>

Цвета для секционной пряжи рассчитываются автоматически с учетом формы детали.
На следующем скриншоте пример секционной пряжи и секционной пряжи с такими же секциями, но с эффектом ручного окрашивания.

<img
  src={require('/img/knitting/section1.png').default}
  srcSet={require('/img/knitting/section1.png').default + " 2x"}
  alt="Секционная пряжа"
/>

<img
  src={require('/img/knitting/section2.png').default}
  srcSet={require('/img/knitting/section2.png').default + " 2x"}
  alt="Секционная пряжа с эффектом ручного окрашивания"
/>

Пример из секционной пряжи на детали сложной формы. При расчете цвета петель из секционной пряжи редактор учитывает длины, цвета и порядок секций, направление вязания в ряду, плотность вязания. В настоящий момент не учитывает протяжки.

<img
  src={require('/img/knitting/section3.png').default}
  srcSet={require('/img/knitting/section3.png').default + " 2x"}
  alt="Рукав из секционной пряжи"
/>

После изменения цвета пряжи, все петли из нее обновляются автоматически.

## Метраж (опционально)

Поле **«Метраж на 100 г»** нужно только для оценки расхода пряжи в панели.  
Оно не влияет на отображение вязания.

---

## Удаление пряжи

Удалить пряжу можно, **только если** в детали **нет ни одной петли** из этой пряжи.
