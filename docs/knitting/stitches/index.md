# Типы петель

В каждом проекте есть **выбранный тип петли** — так же, как выбираются пряжа и плотность.  
Текущий значок петли отображается в панели над холстом:

  alt="Текущий выбранный тип петли"
/>

Нажмите на значок, чтобы открыть палитру и выбрать другой тип:

<img
  src={require('/img/knitting/stichTypes.png').default}
  srcSet={require('/img/knitting/stichTypes.png').default + " 2x"}

## Выбор петли при работе кистью

Когда активен инструмент **Кисть**, палитра петель появляется автоматически — это позволяет сразу выбирать, какой петлёй рисовать:

## Доступные петли

Сейчас доступны базовые типы петель:

- лицевая
- изнаночная  
- накид
- убавки вправо и влево  
- прибавки вправо и влево
- косы

Список постепенно расширяется: планируется добавление более сложных комбинированных петель и специальных обозначений.
